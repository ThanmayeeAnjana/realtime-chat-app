# Realtime Chat App

A real-time chat application built using JavaScript, Firebase Authentication, and Cloud Firestore.

## Features

* User Signup and Login
* Real-time Messaging
* Online User Tracking
* Typing Indicators
* Message Timestamps
* Last Seen Status
* User Search
* Responsive Dark UI
* Firebase Authentication
* Cloud Firestore Database

## Technologies Used

* HTML
* CSS
* JavaScript
* Firebase Authentication
* Cloud Firestore
