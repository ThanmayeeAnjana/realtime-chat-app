# Realtime Chat App

A real-time web-based chat application built using JavaScript, Firebase Authentication, and Cloud Firestore. The application allows users to sign up, log in, and exchange messages instantly with other users.

## Features

* User Registration and Login
* Secure Authentication using Firebase
* Real-Time Messaging
* Online User Tracking
* Typing Indicators
* Message Timestamps
* Last Seen Status
* User Search Functionality
* Responsive Dark-Themed Interface
* Cloud-Based Data Storage with Firestore

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Firebase Authentication
* Cloud Firestore

## Project Structure

```text
auth.js         -> Authentication logic
chat.js         -> Chat functionality
firebase.js     -> Firebase configuration
signup.html     -> Login and Signup page
chat.html       -> Chat interface
style.css       -> Styling
```

## How It Works

1. Users create an account or log in.
2. Authentication is handled by Firebase Authentication.
3. Messages are stored in Cloud Firestore.
4. Real-time listeners update chats instantly for all connected users.
5. Online status, typing indicators, and timestamps enhance the user experience.
